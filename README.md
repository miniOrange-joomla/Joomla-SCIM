# Joomla SCIM User Sync / Provisioning Plugin
Joomla User Management: SCIM Extension

Simplify user management with the **SCIM User Sync/Provisioning plugin for Joomla**, enabling automated user synchronization from your Identity Provider (IDP) to Joomla. This robust tool adheres to the **SCIM (System for Cross-domain Identity Management)** protocol, ensuring real-time updates between your IDP and your Joomla user directory.

## Key Highlights:
- **Automated User Sync**  
  Create, update, delete, or deactivate Joomla users in real-time whenever changes occur in your IDP (Azure AD, Okta, G-Suite, etc.).
- **SCIM Compliant**  
  Converts Joomla into a SCIM-compliant endpoint for seamless integration with any SCIM-enabled IDP.
- **Custom Attribute Mapping**  
  Sync custom user attributes from your IDP and display them directly in the Joomla User Table.

---

## Free Version Features:
- Unlimited user synchronization between IDP and Joomla.
- Real-time provisioning for new and updated users.
- Pre-integrated support for major IDPs including **Azure AD**, **Okta**, **OneLogin**, and **G-Suite**.

---

## Upgraded Version Features:
- **Delete Users**  
  Automatically remove users from Joomla when de-provisioned in your IDP.
- **Deactivate Users**  
  Inactivate Joomla accounts without deleting them, allowing record retention.
- **Update Users**  
  Automatically sync profile changes (e.g., First Name, Last Name) when modified in IDP.
- **Custom Filters**  
  Sync only specific user sets using flexible filtering rules.
- **Custom Attribute Sync**  
  Map additional attributes and display them in Joomla’s user management interface.

---

## Why Use SCIM User Sync?
- Keeps your Joomla user directory fully up-to-date without manual intervention.
- Streamlines provisioning and de-provisioning for employees, contractors, partners, and more.
- Eliminates data mismatches by using your IDP as the **single source of truth** for user information.

---

## Quick Setup:
1. Configure the **SCIM Base URL** and **Bearer Token** in your IDP (Azure AD, Okta, G-Suite, etc.).  
2. Install and set up the SCIM User Sync plugin on Joomla.  
3. Start real-time user synchronization effortlessly.
